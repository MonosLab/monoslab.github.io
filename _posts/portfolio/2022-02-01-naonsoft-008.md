# 메일보관함(웹) 클라이언트
> 기간 : 2022.02 ~ 2022.04

---

**■ 사용 기술**

>  * 역활 : 응용SW개발
>  * 시스템 환경 : Windows
>  * 사용 기술 : C++

메일보관함은 그룹웨어의 메일을 로컬PC에 백업하여 관리할 수 있는 프로그램입니다. UI 구성은 그룹웨어에서 하며, 네이티브 프로그램에서는 트레이 아이콘 외의 화면은 별도로 존재하지 않습니다.
