# (New)아웃룩싱크
> 기간 : 2020.07 ~ 2020.12  
> Outlook Addin 프로그램

---

**■ 사용 기술**

>  * 역활 : 응용SW개발
>  * 시스템 환경 : Windows
>  * 사용 기술 : C++, C#

아웃룩싱크는 그룹웨어의 일정/할일을 로컬PC에 설치된 Outlook의 일정/할일과 동기화 시켜주는 프로그램입니다. 또한 메일 작성시 그룹웨어와 연동하여 조직도를 보여주며, 받는사람, 참고, 숨은참조 등에 조직도내 부서와 사용자를 추가할 수 있습니다.
