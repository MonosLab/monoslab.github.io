# 웹 이미지 편집툴
> 기간 : 2002.03 ~ 2002.04

---

**■ 사용 기술**

>  * 역활 : 응용 SW 개발
>  * 시스템 환경 : Windows
>  * 사용 기술 : C++

---

ActiveX로 구현하여 웹에서 이미지를 편집할 수 있도록 개발함.   
- 이미지 반전
- 이미지 밝기
- 대조(대치)
- Blur, Sharpen, Out-Line(Laplacian), Embossing
- 상하좌우 반전, 좌우 90도 회전
- 이미지 잘라내기
